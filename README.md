Project Overview

Artemia is a mobile application developed to raise awareness and eventually implement detection capabilities for microplastics in food sources. The app's name pays homage to Artemia franciscana, the model organism used in the project creator's Master's thesis to study the genomic impact of microplastic pollution.

Currently, the application focuses on image management and providing essential educational resources. The core function of automated microplastic analysis is scheduled for future implementation.
